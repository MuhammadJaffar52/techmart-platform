# TechMart Platform

A production-grade DevOps learning project that evolves from a local application to a fully automated cloud-native platform.

## Goals

- Build a complete production application
- Learn Docker
- Learn Kubernetes
- Learn Helm
- Build CI/CD pipelines
- Implement GitOps
- Add Observability
- Secure the platform
- Deploy on AWS using Terraform
- Practice real production troubleshooting
